# AI_PROFILE

## Architecture Baseline
- **Backend:** FastAPI (Python) for heavy algorithmic logic & margin calculations.
- **Administrative Layer:** Node.js layer for CRUD routing.
- **Frontend:** Next.js (React) for Server-Driven UI (SDUI) storefronts.
- **Automation:** Local automation pipelines (n8n/PowerShell).

## Local Ports
- FastAPI: `8000`
- Next.js Admin/Storefront: `3000`
- Node.js Layer: `3001`

## Strict Constraints
- **Coding Standards:** Prioritize execution speed, system efficiency, and optimized data syncs (e.g., live MCX bullion feeds via Redis caching).
- **UI/UX Constraints:** Enforce a strict minimalist design. Use clean, plain-text headers. Do not introduce complex logo structures or unnecessary visual elements. Render brand logos strictly as plain text.
- **Metadata Constraints:** NEVER use marketing buzzwords like 'timeless', 'craftsmanship', 'exquisite', or 'heritage' in any generated data, placeholders, or product titles. Use factual parameters (e.g., 18KT, SI-HI).
